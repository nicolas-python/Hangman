# Hangman Game

Ein einfaches Galgenmännchen-Spiel in Python

## Funktionen
- Zufällige Wörter
- begrenzte Versuche
- ASCII-Galgen

## Starten

Starte das Programm mit:

Python hangman.py
Folge den Anweisungen auf dem Bildschirm:
Rate Buchstaben, indem du sie eingibst.
Das Spiel zeigt die bereits geratenen Buchstaben an.
Falsche Buchstaben bringen dich dem Galgen näher.
Das Spiel endet, wenn:
Du das Wort richtig errätst (Gewonnen!)
Oder deine Versuche aufgebraucht sind (Game Over!)
